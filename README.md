# 🔧 GarageFlow — Frontend

Single-page application for **GarageFlow**, a vehicle service management system for garages — track service jobs from intake to delivery.

> 🚧 In active development. Backend lives at [garageflow-api](https://github.com/mehedi-200/garageflow-api).

## Planned Features

- 🔐 Login / logout / profile (Sanctum token auth)
- 👥 Customers — list, search, add, edit, delete
- 🚗 Vehicles — registry with brand, model, year, registration number
- 🛠️ Service jobs board with status workflow: `Pending → In Progress → Completed → Delivered`
- 🧾 Invoices with payment status
- 📊 Dashboard — totals, vehicles in service, revenue

## Tech Stack

- React + Vite
- React Router
- TanStack Query
- Axios
- Tailwind CSS
- React Hook Form

## Git Workflow

`main` ← `develop` ← `feature/*` branches, merged via pull requests.
